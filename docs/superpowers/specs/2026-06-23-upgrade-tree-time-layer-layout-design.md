# 업그레이드 트리 시간층 배치 설계

## 목표

카드 업그레이드 트리를 1시간 데모 기준으로 재배치한다. 플레이어는 10초 플레이, 10초 업그레이드 선택을 반복한다고 가정한다. 5분은 약 15루프이며, 매 5분마다 새 규칙이나 새 플레이 목표를 주는 메이저 카드를 만난다.

이 설계의 목적은 단순 수치 강화가 연속되는 구간을 줄이고, 장비, 수확, 환경 어느 갈래를 타도 주기적으로 큰 성장감을 느끼게 하는 것이다.

## 배치 방식

방식은 시간층 배치로 한다. 세로축은 시간 진행이다. 시작 노드는 아래쪽, 후반 노드는 위쪽에 둔다.

- `y = 0`: 시작
- 이후 5분 단위마다 `y -= 220`
- 같은 시간층의 일반 카드는 같은 `y` 주변에 둔다
- 메이저 카드는 해당 시간층의 중심 카드로 둔다
- 카드 간 최소 간격은 `x 150`, `y 110` 이상으로 둔다

가로축은 역할 갈래다.

- 좌측: 장비
- 중앙: 수확
- 우측: 환경
- 바깥쪽: 특수 열매

좌표 영역:

- 장비 줄기: `x = -450 ~ -1000`
- 수확 줄기: `x = -120 ~ 180`
- 환경 줄기: `x = 450 ~ 1000`
- 특수 열매: `x <= -1150` 또는 `x >= 1150`

## 갈래 정의

### 장비

플레이어가 직접 조작하는 손맛을 바꾸는 줄기다. 낫 성능, 공격 범위, 공격 속도, 발화, 장비 선택, 트랙터 면허가 이 줄기에 속한다.

초반에는 낫이 좋아지는 느낌을 준다. 중반에는 넓은 낫, 빠른 낫, 폭탄 낫 같은 선택지를 열어준다. 후반에는 트랙터 면허처럼 조작 방식 자체가 바뀌는 카드를 둔다.

### 수확

경제와 점수, 운전 실력 보상을 담당하는 중앙 줄기다. 시장 수레, 깔끔한 잔디, 황금 밭, 클리어 보너스, 큰풀 보상, 최종 골드 보정이 이 줄기에 속한다.

중앙 줄기는 가장 안정적인 진행 루트다. 어느 빌드도 중앙 줄기 일부를 타면 경제가 따라오게 한다.

### 환경

맵과 필드 규칙을 바꾸는 줄기다. 바위, 나무, 특수풀, 재성장, 맵 확장, 30m 밭 해금이 이 줄기에 속한다.

환경 줄기는 위험과 보상을 함께 만든다. 장애물은 방해만 하지 않고 파괴 보상, 벌목 연쇄, 새 수확 루트로 이어진다.

### 특수 열매

세 줄기 밖에 달리는 큰 이벤트 카드다. 폭탄, 외계 문양, 레이저, 드론, 번개, 회오리, 트랙터 소환이 여기에 속한다.

특수 열매는 독립 4번째 줄기로 두지 않는다. 관련 줄기의 끝이나 분기점에서 바깥쪽으로 열리게 한다. 예를 들어 폭탄은 환경/장비 사이, 외계 문양과 레이저는 장비/폭탄 이후, 드론과 회오리는 후반 자동 수확 분기로 둔다.

## 5분 단위 메이저 카드 계획

| 시간 | 루프 | 메이저 카드 | 주 체감 |
|---:|---:|---|---|
| 0분 | 0 | 녹슨 낫 갈기 | 시작 카드, 트리 개방 |
| 5분 | 15 | 깔끔한 잔디 | 운전 실력 보상 시작 |
| 10분 | 30 | 불붙은 날, 바위 조사 | 베기에 화상 추가, 장애물 등장 |
| 15분 | 45 | 씨앗 폭탄 | 폭탄과 연쇄 폭발 시작 |
| 20분 | 60 | 황금 밭, 나무 조사 | 초반 러시 보상, 환경 변화 |
| 25분 | 75 | 넓은 밭 해금 | 30m 맵 개방 |
| 30분 | 90 | 넓은 낫, 빠른 낫 | 장비 선택 시작 |
| 35분 | 105 | 외계 문양 | 화면 이벤트형 광역 수확 |
| 40분 | 120 | 트랙터 면허 | 이동 자체가 수확으로 변화 |
| 45분 | 135 | 레이저 | 전방 관통 공격과 폭탄 연계 |
| 50분 | 150 | 드론 또는 번개 | 자동 수확 시작 |
| 55분 | 165 | 회오리 또는 트랙터 소환 | 대형 자동 수확 |
| 60분 | 180 | 후반 강화 묶음 | 1시간 데모 피날레 |

## 카드 연결 규칙

다음 티어를 열기 위해 같은 티어의 모든 카드를 요구하지 않는다. 하위 카드와 직접 연결된 선행 카드만 요구한다.

작은 카드와 큰 카드의 흐름은 아래 규칙을 따른다.

1. 새 경험 카드 1개를 연다.
2. 그 경험을 강화하는 작은 카드 3개에서 6개를 배치한다.
3. 강화 카드 중 1개에서 다음 새 경험 카드로 분기한다.
4. 같은 시간층 안에서 장비, 수확, 환경 중 최소 2개 갈래에 선택지가 보이게 한다.

## 비용 규칙

비용은 5분 단위 목표 시간에 맞춰 조정한다. 정확한 금액은 구현 단계에서 현재 골드 곡선 테스트로 보정한다.

초안 기준:

- 0분: 10g
- 5분: 25g ~ 60g
- 10분: 80g ~ 180g
- 15분: 160g ~ 300g
- 20분: 280g ~ 500g
- 25분: 500g ~ 800g
- 30분: 800g ~ 1300g
- 35분: 1000g ~ 1600g
- 40분: 1400g ~ 2200g
- 45분: 1600g ~ 2600g
- 50분: 2000g ~ 3200g
- 55분: 2600g ~ 4200g
- 60분: 3200g 이상

수확 줄기 비용은 같은 시간층보다 약간 낮게 둔다. 장비 줄기는 평균값으로 둔다. 특수 열매는 같은 시간층보다 높게 둔다.

## 시각 규칙

트리는 식물이 위로 자라는 인상을 유지한다.

- 중앙 수확 줄기는 황금색 느낌으로 단단히 올라간다.
- 장비 줄기는 왼쪽으로 가지처럼 퍼진다.
- 환경 줄기는 오른쪽으로 가지처럼 퍼진다.
- 특수 열매는 바깥쪽에 큰 카드로 붙는다.
- 모든 카드 해금 시 좌우 가지와 중앙 줄기가 함께 완성된 생명수처럼 보인다.

## 구현 범위

이번 작업은 카드 배치와 해금 타이밍 밸런싱에 집중한다.

수정 대상:

- `src/game/config/cards.json`
- 카드 좌표 `layout`
- 카드 티어 `tier`
- 선행 조건 `prereq`
- 비용 `cost`
- 필요 시 카드 표시 순서 `sort`

수정하지 않는 범위:

- 신규 카드 효과 구현
- 신규 시각 이펙트 구현
- 저장 데이터 구조 변경
- 업그레이드 화면 조작 방식 변경

## 검증 기준

구현 후 다음 검증을 통과해야 한다.

1. 모든 카드의 선행 조건이 존재하는 카드 ID를 가리킨다.
2. 루트에서 모든 카드까지 도달 가능하다.
3. 직접 연결된 선행 카드만 있으면 하위 카드가 열린다.
4. 5분 단위 예상 진행에서 최소 1개 메이저 카드가 새로 열린다.
5. 1시간 예상 진행에서 메이저 카드가 10개 이상 열린다.
6. 카드 간 화면 충돌이 없다.
7. 모바일 최저 줌아웃에서 주요 줄기 3개가 구분된다.
8. `npm test`, `npm run build`, `npm run smoke`가 통과한다.

## 리스크

기존 카드 수가 138장이라 한 번에 모든 비용을 정확히 맞추기는 어렵다. 첫 구현은 5분 단위 메이저 카드 도달성을 우선한다. 이후 실제 골드 획득 곡선 테스트 결과로 비용을 2차 조정한다.

한국어 카드 이름 일부가 JSON에서 깨져 보인다. 이번 작업은 `id`, `nameEn`, `effects` 기준으로 배치한다. 표시 이름 정리는 별도 작업으로 분리한다.
